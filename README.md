# Conceal

Conceal is an [Atom][] package that conceals text and replaces it with prettier
alternatives.

![Screenshot][]

To configure Conceal, set the `replacements` option in your config.

``` cson
# config.cson
"*":
  conceal:
    replacements:
      ".": "âˆ˜"
```
